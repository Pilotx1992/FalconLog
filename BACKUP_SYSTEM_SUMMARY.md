## 🔧 **المكونات الرئيسية**

### **BackupService** - الخدمة الرئيسية
```dart
- startAutoBackup(): بدء النسخ الاحتياطي الآلي
- backupToFirebase(): نسخ إلى Firebase
- backupToLocal(): نسخ محلي
- restoreFromBackup(): استعادة البيانات
- performMaintenance(): صيانة تلقائية
- getBackupHistory(): تاريخ النسخ مع cache
```

### **BackupLogger** - نظام التسجيل
```dart
- logSuccess(): تسجيل العمليات الناجحة
- logError(): تسجيل الأخطاء
- logMaintenance(): تسجيل الصيانة
- getRecentLogs(): آخر الأحداث
```

### **NotificationService** - الإشعارات
```dart
- showBackupSuccess(): إشعار نجاح النسخ
- showBackupError(): إشعار فشل النسخ
- showRestoreSuccess(): إشعار نجاح الاستعادة
- showMaintenanceComplete(): إشعار الصيانة
```

### **EncryptionService** - التشفير
```dart
- encrypt(): تشفير AES-256
- decrypt(): فك التشفير
- generateSecureKey(): توليد مفاتيح آمنة
```
